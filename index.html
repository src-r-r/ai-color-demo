AI Color Demo

Generate data → Train a model → Run inference — entirely in your browser.

Step 1: Generate Training Data

AI models learn from examples. This step creates a grid of colored tokens that the model will study to learn patterns. Each color is assigned an index (0–5), and the model will try to predict which color comes next based on the ones before it.

Pick a pattern to define how colors relate to each other. The noise slider adds randomness—more noise makes patterns harder to learn.

Pattern The rule governing which colors follow which. The model will learn this rule.
Noise Randomness injected into the pattern. Low = clear patterns, high = harder to learn.
Rows / Cols Size of the training data. More data = more training examples for the model.
Transition Matrix Visual map of the pattern: darker cells = more likely transitions from row color to column color.

Step 2: Train the Model

The model is a simple neural network (LSTM) that learns to predict the next color from a window of previous colors. Training adjusts the model's internal weights so its predictions get closer to the actual data.

As training progresses, loss should decrease (the model is making fewer mistakes) and accuracy should increase (more correct predictions).

Context Length How many previous colors the model sees. Longer context = more information, but harder to learn.
Epochs How many times the model cycles through all the training data. More epochs = more learning opportunity.
Learning Rate How big each weight adjustment is. Too high = overshoots; too low = learns too slowly.
Loss Chart Shows loss over time. A downward trend means the model is learning. A flat line means it has plateaued.
Epoch — current pass through the data
Loss — average error (lower is better)
Accuracy — % of correct predictions

Step 3: Run Inference

Now that the model has learned from the training data, you can see what it predicts. Give the model a sequence of colors, and it will guess the next one based on the patterns it learned.

Each time you click Predict Next, the model's prediction is appended to the sequence. The next prediction uses the updated sequence, so the model is essentially continuing the pattern it has learned.

Sequence Length Length of random sequences generated by "Random Sequence".
Predict Next Feeds the current sequence to the model and appends the predicted next color.
Input Sequence The colors the model has been given so far. Build it manually or use random generation.
Prediction Shows the full sequence with the latest predicted token highlighted (accent border).
